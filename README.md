# HelloPod

A modern, responsive web application delivering a professional user experience with clean design and smooth interactions.

## Features

- Clean, professional design with smooth animations
- Responsive layout across all devices
- Interactive user interface with modern JavaScript
- Multi-page architecture for different use cases
- Optimized media handling with background videos
- Comprehensive web pages covering various business scenarios

## Pages

- **index.html** - Main landing page
- **booking.html** - Booking/Appointment interface
- **capabilities.html** - Product/service capabilities showcase
- **cases.html** - Case studies and success stories
- **confirmation.html** - Confirmation/thank you page
- **industries.html** - Industries served section
- **solution.html** - Solution and features overview

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Modern CSS Flexbox/Grid layouts
- CSS animations and transitions

## Getting Started

Clone the repository and open any HTML file in your browser:

```bash
git clone https://github.com/vaibhav410/HelloPod.git
```

Then simply open `index.html` in any modern web browser.
